Testing

Introduction

Here we have a little macro to rerun arbitrary code multiple times in case it failes for some reason. It is used to retry testing of downloading from osm, since the server often is not willing to serve a request on the first try.

API

  • CoolWalksUtils.@rerun
CoolWalksUtils.@rerun — Macro
@rerun(number, code)

runs the input code at most number times or until it does not throw an error. Used to test code which talks to a server which may not respond on every call.
